## 🏗️ 架构改进

### 统一的插件接口
```rust
pub trait Plugin: Send + Sync + Debug {
    fn name(&self) -> &'static str;
    fn metadata(&self) -> PluginMetadata;
    async fn handle_request(...) -> Result<(bool, Option<HttpResponse>)>;
    async fn handle_response(...) -> Result<bool>;
    async fn start(&mut self) -> Result<(), PluginError>;
    async fn stop(&mut self) -> Result<(), PluginError>;
}
```

### 优先级系统
```
RequestId: 10      (最早执行)
OAuth2: 90         (认证)
BasicAuth: 100     (认证)
LlmRouter: 200     (路由)
PromptTransformer: 300 (转换)
```
